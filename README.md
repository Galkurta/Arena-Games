- Auto claim
- Auto use boost (bug for multi account)
- Multi account
- Register Arena Games [here](https://t.me/Arenavsbot?start=ref_gUYacj83JULbhjsyo3wVRY)

## Installation
```shell
git clone https://github.com/Galkurta/Arena-Games.git
```
```shell
cd Arena-Games
```
```shell
npm install
```
## Usage

### Edit data.txt
```shell
nano data.txt
```
### Create screen
```shell
screen -S your_screen
```

### Run
```shell
node main.js
```
